The $0 AI Stack: 6 Tools That Replace a £50k Software Budget

Perplexity, Cursor, Gamma, Claude, Leonardo, Phind — six tools that are completely free, professionally powerful, and the exact stack used to run this site and build a real income from AI.

Two years ago, building a professional AI-powered content and income operation required expensive subscriptions, a developer, and a design team. Today, a student with a laptop and a WiFi connection has access to tools that match or exceed that entire setup — for free.

This is a breakdown of the six tools in the AI Architect stack. Not tools I tested once and listed — tools used daily, in production, to research, write, code, visualise, and distribute content at scale.

How to use this guide: Each tool profile below includes what it does, the exact use cases it covers in this stack, and a direct link. Read all six before deciding where to start — they work best as a system, not in isolation.

The Stack

🔍
Perplexity AI
// ROLE: Research Engine

Perplexity is what Google should be. It doesn't just return links — it reads the top sources, synthesises the information, and delivers a structured answer with cited references you can verify. For anyone producing research-based content, it eliminates the most time-consuming part of the process.

In this stack it handles all topic research, competitor analysis, niche discovery, and fact-checking. A 40-minute research session becomes a 4-minute Perplexity query with better source coverage.

Content research Fact-checking Niche discovery Literature review Competitor analysis
Get Access → perplexity.ai
⌨️
Cursor Editor
// ROLE: Coding Without Coding

Cursor is a code editor built around AI assistance. You describe what you want in plain English, and Cursor writes the code, runs it, reads any error messages, and fixes them — in a loop until it works. For automation scripts, data tools, and micro-applications, it removes the coding barrier entirely.

In this stack it handles all custom automation scripts, data processing, and any client deliverables that require code. No prior programming knowledge required — just the ability to describe a problem clearly.

Automation scripts Data processing Web scraping API integrations Custom tools
Get Access → cursor.com
⚡
Gamma App
// ROLE: Visual Content Engine

Gamma takes text — notes, bullet points, a blog post URL — and turns it into a polished, professional presentation or visual document in under 60 seconds. The design quality is genuinely impressive, and the output is shareable and embeddable immediately.

In this stack it handles all client-facing deliverables, social content (especially Pinterest), and the visual version of every blog post. It's the difference between a wall of text and something people actually share.

Presentations Pinterest visuals Client decks Report design Content repurposing
Get Access → gamma.app
🤖
Claude 3.5
// ROLE: Primary AI Brain

Claude is the primary writing, reasoning, and analysis engine in this stack. It produces longer, more coherent outputs than most alternatives, handles nuanced instructions exceptionally well, and is the best model available for producing content that doesn't read like it was AI-written.

In this stack it handles all drafting, editing, summarising, extraction, and analysis tasks. It's the tool open in a browser tab every single day.

Content drafting Document analysis Summarisation Email drafting Strategic thinking
Get Access → claude.ai
🎨
Leonardo AI
// ROLE: Visual Asset Generator

Leonardo is a free AI image generation platform with a generous free tier and output quality that matches paid alternatives. For blog cover images, Pinterest graphics, social media assets, and content thumbnails, it eliminates the need for a designer or a stock photo subscription.

In this stack it handles all custom imagery — header images, illustrations for posts, and Pinterest-optimised visuals that stand out in a feed.

Blog images Pinterest graphics Social assets Thumbnails Brand visuals
Get Access → leonardo.ai
🧠
Phind
// ROLE: Technical Search Engine

Phind is a search engine built specifically for technical and programming queries. Where Google returns Stack Overflow threads from 2014, Phind reads multiple current sources and gives you a direct, working answer. For anyone building with code — even using Cursor — it's the fastest way to solve technical problems.

In this stack it handles all technical troubleshooting, code debugging questions, and implementation research when building automation workflows.

Code debugging Technical docs API questions Implementation help
Get Access → phind.com

How the Stack Works Together

The power isn't in any individual tool — it's in the workflow they form together. Here's how a typical content piece moves through the stack:

  • Perplexity researches the topic and surfaces the key angles
  • Claude drafts the post structure, then writes the full piece
  • Gamma turns the post into a Pinterest-ready visual and a shareable deck
  • Leonardo generates the cover image and any supporting visuals
  • Cursor handles any custom scripts needed (scheduling, formatting, data)
  • Phind solves any technical issues that arise along the way

Total cost of this stack: £0. Total time from blank page to published post with visuals: under 30 minutes.

Start with two: If you're new to this, don't try to use all six tools at once. Start with Perplexity and Claude. Get comfortable with that research-to-draft workflow. Add Gamma when you want to create social content. Build from there.
